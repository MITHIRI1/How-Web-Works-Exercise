What is HTTP? The Hypertext Transfer Protocol is a protocol for fetching resources such as HTML documents. It is the foundation of any data exchange on the Web and it is a client-server protocol. What is a URL? Uniform Resource Locatorsis the mechanism used by browsers to retrieve any published resource on the web. What is DNS? Domain Name System purpose is to translate a domain name into the appropriate IP address. What is a query string? A query string is the portion of a URL where data is passed to a web application and/or back-end database. What are two HTTP verbs and how are they different? GET Retrieve whatever information is identified by the Request-URI POST Request that the destination server accept the entity enclosed in the request as a new subordinate of the resource identified by the Request-URI What is an HTTP request? A HTTP request is made by a client to a named host, which is located on a server. What is an HTTP response? A HTTP response is made by a server to a client. The aim of the response is to provide the client with the resource it requested What is an HTTP header? Give a couple examples of request and response headers you have seen. Let the client and the server pass additional information with an HTTP request or response. One example is the Cache-Control general-header field is used to specify directives that MUST be obeyed by all the caching system.
